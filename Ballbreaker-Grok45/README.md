# Ball Breaker

2.5D Arkanoid-style brick breaker built with Three.js.

## Play

```bash
npm install
npm run dev
```

Open the URL Vite prints (default `http://localhost:3000`).

## Single-file build

```bash
npm run build
```

Output is a single self-contained HTML file at `dist/index.html` (Three.js inlined, no external deps).

## Controls

- **Mouse / touch** — move paddle
- **← →** or **A D** — move paddle
- **Click / tap / Space** — launch ball

## Screens

1. Startup
2. Gameplay (score, lives, level HUD)
3. Game over / win
